## Run Server

First, run:

### `cd trip-server`

Install Dependencies:

### `npm install`

Then:

### `node index.js`

## Run Project

First, run:

### `cd chooose-submission`

Install Dependencies:

### `npm install`

Then to run the project:

### `npm start`

To run the tests:

### `npm test`
